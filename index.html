What the logger do?

The logger is a multi-channel logger that allows you to manage your logs of your application with ease. It supports multi destination like Console, Remote, File with configurable log rolling based on file size or date.

Multi channel

Support multi destination like Console, Remote, File with configurable log rolling based on file size or date.

Plugins based

Use decorators to declare your own appenders and layouts logger.

Colored

Colored console logging to stdout or stderr.
